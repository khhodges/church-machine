Church Capability Extensions

                    

                        
                            

                                Mnemonic
                                Format
                                Description
                                Required Permissions
                                Encoding
                            

                        
                        
                            

                                CAP.LOAD
                                custom-0
                                Load capability from C-List into CR
                                L (Load)
                                funct7 --- rs1 000 cr_dst 0001011
                            

                            

                                CAP.SAVE
                                custom-0
                                Save capability from CR to C-List
                                S (Save)
                                funct7 cr_src rs1 001 --- 0001011
                            

                            

                                CAP.CALL
                                custom-0
                                Protected call through capability, save return state
                                E (Enter)
                                funct7 cr_src rs1 010 cr_dst 0001011
                            

                            

                                CAP.RETURN
                                custom-0
                                Return from protected call, restore context
                                (none)
                                funct7 ----- --- 011 ----- 0001011
                            

                            

                                CAP.CHANGE
                                custom-0
                                Change thread context, create new thread GT in CR8
                                M (Machine)
                                funct7 cr_src rs1 100 ----- 0001011
                            

                            

                                CAP.SWITCH
                                custom-0
                                Copy capability to system register CR8-CR15 (target in bits [24:22])
                                M (Machine)
                                tgt cr_src rs1 101 ----- 0001011
