*** Errata ***


*** Upgrading from previous Efinity versions ***

Previously generated data files from Efinity compilations (such as old 
synthesis, place and route output files) may not be compatible with this
version of Efinity software.  Full recompilation is recommended.


*** Description ***

This is a release package of the Efinity tools for compiling and simulating
designs targeting the Efinix programmable logic devices.


*** New features in Efinity 2025.2 ***

   -- [Titanium] Adds (preliminary) Q3 automotive timing models for the
       following devices:
       -- Ti135N484
       -- Ti375N484
   -- [Titanium] Ti135N441, Ti85N441 devices support up to PCI-e Gen4x4
   -- Reduces compile times and memory usage ~30% for all device families
   -- Significantly reduces Efinity project compiled database file sizes,
      including files created in outflow and work_* project directories.
   -- Improves speed and responsiveness of Efinity Floorplan Viewer
   -- Adds SDC timing constraint support for "set_bus_skew" and
      "report_bus_skew" commands
   -- Package Viewer is now Package Planner, with ability to drag and drop
      pin assignments and other features
   -- Generates hierarchical placement report during place and route tool
   -- Improves handling of Efinity compiler messages
       -- Adds the ability to suppress info or warning messages by ID
       -- Documents more compiler messages in integrated help
   -- Improves support for Efinity's unified design flow
       -- Allows user to instantiate MIPI_LN_RX, MIPI_LN_TX interface blocks in
          user's design RTL
       -- Unified simulation support for nearly all supported interface blocks,
          including large blocks such as Titanium/Topaz LPDDR4 interface block,
          PCI-e block, etc.  [Note that encrypted simulation models for some
          blocks not included with Efinity, can be obtained separately.]
   -- New example designs targeting various development kits included in
      <installation path>/examples folder
   -- [Titanium / Topaz] Transceiver debugger now includes bathtub view
   -- Includes new OpenOCD-based JTAG Co-debug tool, that allows concurrent
      debugging from multiple tools, including Efinity Debugger concurrent
      with RISC-V IDE debugger
