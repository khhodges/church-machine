*** Description ***

This is a software patch for the Efinity 2025.2 release.

It adds functionality and addresses the following issues found in the
2025.2 release.

*** 2025.2.288.4.# patch notes
    -- [Titanium] Enables bitstreams for Ti135N576D2F4 [DEVINFRA-1148]
    -- [Titanium] Adds I4, I4L timing models to various Ti85, Ti135, Ti165,
       Ti240, Ti375 devices [DEVINFRA-1159]
    -- (Windows only) Improves Programmer and Debugger tools libusb driver loading,
       particularly in regards to new Efinix-built libusb driver for FTDI *232H
       USB controllers [DEVINFRA-1124]
       -- New driver can be downloaded from Efinix support web site, and users
          are encouraged to use this instead of the Zadig tool's libusb driver
    -- Updates Programmer's prebuilt "SPI Active using JTAG Bridge" flash loader images,
       with focus on improved reliability at JTAG speeds up to 30 MHz [PROG-698]
    -- [Titanium, Topaz] Fixes issues modeling HSIO DLY_ENA and DLY_INC dynamic
       delay control ports (can be used across several LVDS, MIPI, GPIO primitives)
       [PT-3232]
       -- Removes "-clock_fall" option in related generated PT.SDC constraints
       -- Updates HSIO block simulation models to sample on rising edges only
    -- Fixes synthesis name preservation issue that could cause wizard Debugger
       flow to fail [SYN-1062]
    -- [Titanium, Topaz] Fixes SGMII APB reset state issue [PT-3195]
    -- Fixes Debugger issue handling 1-bit signal buses [HD-503]

*** Setup Requirements ***

This patch release requires Efinity 2025.2 to be installed.


       
*** Patch Installation ***

Linux patch installation:

        1. Unzip the efinity patch into any temporary directory.
                > unzip efinity-<version>-patch.zip
        2. Setup environment variables
                > source /path/to/efinity/2025.2/bin/setup.sh
        3. Run patch installer
                > cd efinity-<version>-patch
                > ./run.sh
        4. Follow on screen prompts to complete installation.



Windows patch installation:

        1. Unzip the efinity patch into any temporary directory.
                a) From CMD shell
                        > unzip efinity-<version>-patch.zip
                or b)  double-click on patch from Windows Explorer and "Extract all"
        2. Open CMD shell
        3. Setup environment variables
                > \path\to\efinity\2025.2\bin\setup.bat
        3. Run patch installer
                > cd efinity-<version>-patch
                > run.bat
        4. Follow on screen prompts to complete installation.
#!/bin/bash

# Efinity Patch Installer
# 
# INSTALLATION
#
#  1.  In terminal, run this command first, to set up environment variables
#             > source /path/to/efinity/<version>/bin/setup.sh
#  2.  Then run
#             > ./run.sh

silent_python_arg=""
silent=0
no_python=0
while :; do
        case $1 in
                -s|--silent)
                        silent=1
                        silent_python_arg=$1
                        ;;
                -n|--no_python)
                        no_python=1
                        ;;
                -ns|-sn)
                        silent=1
                        silent_python_arg="-s"
                        no_python=1
                        ;;
                -?*)
                        echo "WARNING: Unknown option (ignored): $1" >&2
                        ;;
                *)
                        break
        esac
        shift
done

if [ -z "$EFINITY_HOME" ]; then
        echo "EFINITY_HOME environment variable not set, please run"
        echo "'source /path/to/efinity/<version>/bin/setup.sh'"
        echo "before running this installer"
        exit 1
fi

if [ $silent -ne 1 ]; then
        echo "Efinity installation at ${EFINITY_HOME} will be modified"
        read -p "Is this OK? (Y/N) " -n 1 -r
        echo
        if [[ ! $REPLY =~ ^[Yy]$ ]]; then
                echo "Patch installation aborted by user"
                exit 1
        fi
fi


PATCH_HOME="$(cd "$(dirname "$BASH_SOURCE")" && pwd)"
if [ $no_python -eq 0 ]; then
        "${EFINITY_HOME}/bin/python3" "${PATCH_HOME}/update_manifest.py" $silent_python_arg
        exit_code=$?
        if [ $exit_code -ne 0 ]; then
                if [ $silent -ne 1 ]; then
                        echo "Patch validation failed with error code $exit_code"
                fi
                exit $exit_code
        fi
fi

cp -rf "${PATCH_HOME}"/files/* "${EFINITY_HOME}/"
cp -f "${PATCH_HOME}"/patch_readme.txt "${EFINITY_HOME}/"
patch_removes_file="$(dirname "$0")/patch_removes.txt"
while IFS= read -r line; do
    # Check if the file exists and delete it
    if [ -n "$line" ]; then
        if [ -e "$EFINITY_HOME/$line" ]; then
            rm "$EFINITY_HOME/$line"
        fi
    fi
done < "$patch_removes_file"

echo "Patch installation completed"
